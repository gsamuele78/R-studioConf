# RStudio Open Source Documentation (Intro)

This document provides an introduction to RStudio Open Source Software (OSS) and pointers for getting started with RStudio OSS in R environments.

Key resources

- RStudio Open Source: <https://www.rstudio.com/products/open-source/>
- RStudio OSS documentation: <https://docs.rstudio.com/>
- Quick start with R and RStudio OSS (local setup, RStudio Desktop, or server)

Getting started tips

- Install R packages via install.packages("pkgname")
- Explore RStudio's OSS examples and tutorials
- Combine RStudio OSS with R by running R scripts in RStudio or through the command line
