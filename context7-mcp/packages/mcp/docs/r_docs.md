# R Documentation (Intro)

This document provides introductory information and quick links for R, a powerful language and environment for statistical computing and graphics.

## Official Resources

* **R Project Website:** [https://www.r-project.org/](https://www.r-project.org/)
* **R Manuals:** [https://cran.r-project.org/manuals.html](https://cran.r-project.org/manuals.html)
* **R Task Views:** [https://cran.r-project.org/web/views/](https://cran.r-project.org/web/views/) - Curated lists of packages for specific fields.

## Key Concepts

* **Vectors:** The fundamental data structure in R for storing elements of the same type.
* **Data Frames:** Two-dimensional, tabular data structures with columns of potentially different types.
* **Packages:** Collections of R functions, data, and compiled code that extend R's capabilities.
* **RStudio:** An integrated development environment (IDE) for R, widely used for R development.

## Getting Started

For beginners, it's recommended to start with the official R manuals and explore RStudio for a streamlined development experience.
